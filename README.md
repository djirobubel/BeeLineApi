# Техническое задание для тестового задания на позицию ASP.NET стажера

## Требования

### Функциональные требования:
*Регистрация пользователя*: Возможность регистрации новых пользователей.
*Авторизация*: Логин пользователей с использованием JWT для управления сессиями.
*Просмотр профиля пользователя*: Доступ к профилю по URL.
*Управление друзьями*: Добавление, удаление из друзей, просмотр списка друзей, просмотр детальной информации о друге, изменение статуса (близкий друг или нет).

### Нефункциональные требования:
*Используемые технологии*: ASP.NET Core, Entity Framework Core, (приветствуется использование Docker и MS SQL Server. для задания со звездочками).
*Качество кода*: Код должен быть чистым и понятным, с комментариями, где это необходимо.
*Паттерны проектирования*: Применение стандартных паттернов проектирования (например, Repository Pattern, Dependency Injection).


## Инструкции по запуску:
1. appsettings.json - указать строку подключения к базе MS SQL.
2. Применить миграции Update-Database через Package Manager Console.
3. Запустить проект.
